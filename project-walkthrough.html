Basic Tutorial: Your First Website

Let's create your first professional website step by step. We'll build a tech startup landing page that you can customise for any business.

Step 1: Choose Your Design Style

1 Open Design Generator Pro

Navigate to the Design tab (should be selected by default). You'll see 6 design styles:

  • Minimalist: Clean and simple (great for portfolios)
  • Dashboard: Professional admin-style (perfect for SaaS)
  • Corporate: Traditional business look
  • Creative: Artistic and bold (agencies, artists)
  • Tech Startup: Modern and innovative ← We'll use this!
  • E-commerce: Product-focused retail design

Click on Tech Startup to select it.

2 Select Layout Template

Choose from 5 layout options:

  • Hero: Large impact section at top ← Perfect for startups!
  • Centered: Everything centered on page
  • Sidebar: Navigation on the side
  • Grid: Card-based layout
  • Split: Two-column design

Click on Hero template.

3 Customize Your Content

Fill in your website information:

  • Site Title: "TechFlow Solutions" (or your company name)
  • Subtitle: "Revolutionize Your Workflow with AI-Powered Tools"
  • Navigation Links: Keep the default or customise:
    Home About Features Pricing Contact
  • Main Content: "Our innovative platform combines advanced analytics, automation, and intelligent insights to help you stay ahead of the competition in today's fast-paced digital landscape."
4 Choose Colors and Typography

Customize the visual style:

  • Color Scheme: Select "Professional Blue" (perfect for tech)
  • Typography: Choose "Modern Sans" for clean readability

Watch the preview update in real-time as you make changes!

Design Tip: The preview shows exactly how your website will look. Use the device buttons (Desktop/Tablet/Mobile) to see how it appears on different screen sizes.

Step 2: Add Professional Components

5 Switch to Components Tab

Click the Components tab at the top. You'll see a library of professional components organized by category:

  • Headers: Hero sections, navigation bars
  • Content: Text blocks, testimonials
  • Features: Feature grids, pricing tables
  • Forms: Contact forms, newsletters
  • Footers: Complete footers with links
6 Drag and Drop Components

Let's build a complete startup landing page:

  1. Drag Hero Section from Headers → Drop it in the canvas
  2. Drag Feature Grid from Features → Drop below the hero
  3. Drag Pricing Table from Features → Drop below features
  4. Drag Testimonial from Content → Drop below pricing
  5. Drag Contact Form from Forms → Drop at the bottom
  6. Drag Footer from Footers → Drop at the very bottom
7 Edit Component Content

Click on any component to see edit tools ( ). Click the edit button to customise:

  • Hero Section: Update title to your company name
  • Feature Grid: Edit features to match your services
  • Pricing Table: Adjust plans and prices
  • Testimonial: Add real customer feedback
  • Contact Form: Customize form fields

Step 3: AI-Powered Content Generation

8 Use AI Tools Tab

Click the AI Tools tab for professional content generation:

  1. Select Industry: "Technology & Software"
  2. Choose Content Type: "Hero Section"
  3. Add Keywords: "automation, productivity, AI" (optional)
  4. Click "Generate Content"
  5. Review the AI-generated content
  6. Click "Apply to Current Component" to use it

The AI will generate professional, industry-specific content that converts visitors into customers!

Important: Always review AI-generated content and customise it to match your specific business. The AI provides excellent starting points, but personal touches make it perfect.

🤖 Claude Code Integration

Now comes the exciting part - taking your design and turning it into a real website with Claude Code!

Understanding the Integration Process

Design Generator Pro creates HTML templates that Claude Code can transform into any framework or technology you need. Think of it as giving Claude Code a professional blueprint to work from.

Step 1: Export Your Design

1 Go to Export Tab

Click the Export tab and choose your format:

  • Complete HTML: For basic websites or WordPress
  • React Components: For React.js projects
  • Next.js Project: For Next.js applications (great for Vercel!)
  • Vue Components: For Vue.js projects
  • Nuxt.js Project: For Nuxt.js applications
  • Design Tokens: For design systems
2 Choose Export Options
  • CSS Framework: Vanilla CSS, Tailwind, Bootstrap, or Bulma
  • Include Comments: (helpful for learning)
  • Include Documentation: (creates README files)

Click "Download Design" - you'll get multiple files!

Step 2: Setting Up Claude Code Project

3 Create Project Folder

On your computer, create a new folder for your project:

Navigate to your project folder and start Claude Code. You can now use specific commands:

🌐 Vercel Deployment Guide

Vercel is the best hosting platform for modern websites. It's free, fast, and works perfectly with designs from Design Generator Pro.

Why Vercel?

  • Free Hosting: No cost for personal and small business sites
  • Automatic HTTPS: Secure websites by default
  • Global CDN: Fast loading worldwide
  • Easy Deployment: One-click deploy from GitHub
  • Custom Domains: Use your own domain name
  • Built for Next.js: Perfect integration with Next.js projects

Step-by-Step Vercel Deployment

1 Prepare Your Project with Claude Code

Use this exact command to prepare for Vercel deployment:

Create a Next.js 14 project from my hero-template.html with: - Complete Vercel deployment configuration - Environment variables setup for production - Optimized build configuration - SEO meta tags and sitemap - Error pages (404, 500) - Performance optimisations - GitHub repository setup with proper .gitignore Make it production-ready for Vercel hosting.

Claude Code will create all necessary files and configurations!

2 Push to GitHub

Claude Code will help you set up Git. Use these commands in your project folder:

git add . git commit -m "Initial commit: Design Generator Pro template" git push origin main

Your code is now on GitHub and ready for Vercel!

3 Deploy to Vercel
  1. Go to vercel.com and sign up (free)
  2. Click "New Project"
  3. Connect your GitHub account
  4. Select your project repository
  5. Click "Deploy"

That's it! Vercel automatically detects Next.js and deploys your site.

4 Custom Domain (Optional)

To use your own domain name:

  1. In Vercel dashboard, go to your project
  2. Click "Domains" tab
  3. Add your domain name
  4. Follow the DNS instructions

Your website will be live at your custom domain!

💡 Vercel Pro Tip: Vercel automatically rebuilds your site every time you push changes to GitHub. This means you can update your website just by making changes and pushing to GitHub - no manual deployment needed!

Common Vercel Configurations

Environment Variables

For sites that need API keys or database connections:

  1. In Vercel dashboard, go to your project
  2. Click "Settings" → "Environment Variables"
  3. Add variables like STRIPE_SECRET_KEY, DATABASE_URL
  4. Redeploy your project
Form Handling

For contact forms and submissions, ask Claude Code:

Add serverless form handling to my contact form with: - Vercel serverless functions - Email notifications via SendGrid or Resend - Form validation and spam protection - Success/error message handling - Database storage for submissions (optional) Make the contact form fully functional on Vercel.

Troubleshooting & FAQ

Common questions and solutions for smooth project development.

Common Issues & Solutions

Design Generator Pro Issues:

Problem: Components won't drag and drop
Solution: Refresh the page and try again. Make sure you're on the Components tab and dragging to the canvas area (not the preview area).
Problem: Export downloads multiple files
Solution: This is normal! You get the main HTML, CSS file, design tokens, and documentation. Keep all files together.
Problem: AI content doesn't match my business
Solution: Select the correct industry, add specific keywords, and edit the generated content to match your needs.

Claude Code Integration Issues:

Problem: Claude Code doesn't understand my template
Solution: Be more specific in your request. Mention the exact file name and what you want:
Convert my hero-template.html (attached) into a Next.js project with TypeScript and Tailwind CSS. Keep the exact visual design and layout.
Problem: Colors don't match the original design
Solution: Include the design-tokens.json file and ask Claude Code to use it:
Use the colours and styling from design-tokens.json to ensure the React components match the original template exactly.

Vercel Deployment Issues:

Problem: Build fails on Vercel
Solution: Ask Claude Code to fix build issues:
Fix my Next.js project for Vercel deployment - ensure all dependencies are correct, remove any build errors, and add proper TypeScript configurations.
Problem: Images don't load on Vercel
Solution: Use Next.js Image component:
Convert all images in my project to use Next.js Image component for proper optimisation and loading on Vercel.
